#  Email Tracking System

A complete email tracking system that sends unique links to recipients and tracks who clicks them.

##  Features

- Send personalized emails with unique tracking links
- Track who opens the email
- Track who clicks the link
- Real-time dashboard with filters
- Add recipients manually
- Delete recipients
- Beautiful thank-you page for recipients
- SQLite database for storing data

## Tech Stack

- **Backend:** Node.js, Express
- **Database:** SQLite3
- **Email:** Nodemailer (Gmail SMTP)
- **Frontend:** HTML, CSS, JavaScript
